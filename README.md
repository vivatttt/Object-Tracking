### Getting started
Below is the sequence of actions for launching a project 
```
python -m venv venv
```
```
venv/Scripts/activate
```
Then you need to add all the necessary libraries
```
pip install -r requirements.txt
```
And finally launch the *Object Tracker 3000 (!!)*
```
python -m run
```
